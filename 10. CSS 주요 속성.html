주요속성

외곽선 관련

  • border : 1px solid red(대표속성) / 선설정
  • border-radius : 10px 0px 10px 0/ 둥근모서리
  • border-color : green
  • border-bottom/top/left/right : 1px solid purple

글자관련

  • color : red / 글자색
  • font-size : 20px / 글자크기
  • font-family : 궁서 / 서체
  • font-weight : bold, 100~900 / 글자두께
  • font-style : italic,, normal / 이탤릭체
  • font : bold italic 20px/40px 궁서
  • line-height : 200px / 줄간격
  • text-indent : 20px / 들여쓰기,내어쓰기
  • text-shadow : 2px 2px 0 red / 글자그림자
  • user-select : none / 드래그 방지
  • letter-spacing : 3px / 자간
  • text-align : center, left, right, justify / 문단정렬

배경

  • background-color : green / 배경색
  • background-image : url ()
  • backgroud-size : 100%, 100px, cover, contain
  • background-repeat : no-repeat
  • background-position : left, right, top, bottom, 100px, 50%
  • background : center / cover url() / 배경의 대표속성

크기관련

  • width : 50% /폭
  • height : 100px /높이
  • min-width/max-width : 200px / 최소폭,최댓값
  • min-height/max-height : 최소폭, 최댓값

여백관련

  • margin : 10px / 외곽선 바깥의 여백
  • padding : 10px / 외곽선 안쪽의 여백

display 속성

  • display : inline / 글자속성, span, strong, a... / 크기조정 X
  • display : block / 블록속성, li, ul, nav, haeder, aside, article, div 등 레이아웃 형태
  • display : inline-block / 인라인블록속성, img, table, input.../ 크기조정, 글자정렬

위치관련

  • position:static : (기본)
  • position:absolute : left/right, top/bottom 좌표값으로 이동, layer속성을 가지며 일반 태그보다 상위로 올라간다
  • position:relative : 자손의 position값 기준좌표로 지정
  • position:fixed : 기본적으로 absolute속성 + 스크롤 고정
