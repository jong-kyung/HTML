HTML5의 종류

  • h1 : 페이지의 제목
  • h2~h6 : 소제목
  • p : 글, 문장, 문단
  • img : 이미지
  • a : 링크(하이퍼텍스트)
  • video : 비디오
  • audio : 오디오
  • ul, dl, ol : 리스트
  • table : 표

시멘틱 태그(요소)

  • 웹표준에서는 의미에 맞는 시멘틱 태그를 적극 활용하는 것을 권장
  • header : 홈페이지 상단, 로고, 상호명, 사이트명, 메뉴...
  • footer : 홈페이지 하단, 기타메뉴, 문의사항, 연락처/주소, 카피라잇...
  • main : 주콘텐츠 영역
  • nav : 메뉴(메인메뉴)
  • aside : 부가콘텐츠, 사이트편 한쪽편향된 레이아웃
  • section / article : 큰 콘텐츠그룹 / 작은 단위의 독립콘텐츠
  • adress : 카피라잇, 주소, 연락처
  • strong / em : 글자/문장을 강조할 때
  • details : 자세히보기 기능
  • summary : 자세히보기의 제목을 설정

폼문관련 태그

  • form : 사용자의 정보를 입력받아 서버로 전송하는 기능의 태그
  • fieldset : 입력정보의 그룹을 시각적으로 표현
  • legend : 양식의 제목
  • button : 버튼
  • input : 사용자에게 정보를 입력받는 요소
