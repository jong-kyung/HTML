CSS기본

  • Cascading Style Sheets의 약자
  • HTML문서에서 디자인을 꾸미는 모든 활동
  • 인라인방식 : 태그에 속성으로 직접 작성, 비표준이므로 사용을 비권고
  • 내부스타일 : style 태그로 작성, 비표준
  • 외부스타일 : link 태그로 불러와서 적용하는 방식
  • CSS속성은 기본적으로 부모(조상)에서 자식으로 상속을 받는다

CSS 선택자

  • * : 전체선택자
  • 태그: 요소선택자
  • '스페이스바': 자손선택자
  • > : 자식선택자
  • # : 아이디선택자
  • . : 클래스선택자
  • nth-child() : 가상선택자(형제번호)
  • [태그의속성] : 속성선택자
  • , : 그룹선택자
  • + : 형제선택자
  • ~ : 앞 선택자의 동생들을 선택

CSS 색상표현의 종류

  • 색상명: red, green...
  • 웹색상코드(16진수) : 0~9 + A~F, 0~255(RGB)
  • rgb : rgb(255,0,0)
  • rgba : rgba(255,0,0,1)
  • 나는 첫번째두번째
  • 나는 첫번째두번째
  • 나는 첫번째두번째
