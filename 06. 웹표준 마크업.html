웹표준 마크업

  • 시멘틱태그(요소)적극적으로 활용(header, nav, ul...)
  • 콘텐츠에는 항상 제목을 달아주는 것을 권장
  • 로고는 대부분 페이지 제목으로(h1), h1은 반드시 한 개만 있어야 함, 소스의 순서(위치)는 상관이 없음
  • 사용자 입력폼(폼문), form, fieldset, legend를 필수
  • 디자인 요소는 생략, 기능/콘텐츠 중심으로 작성
  • 링크는 a, 페이지 이동없이 작동하는 것은 button
  • html validator(마크업 유효성 검사) : https://validator.w3.org/, 최소 error 없이
  • 의미가 없는 디자인용 태그 : div, span
