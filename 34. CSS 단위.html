모바일웹

  • 모바일 전용페이지 : 방문자의 접속 디바이스별로 리다이렉트 시켜주는 방식, 모바일 전용으로 최적화 가능, 유지보수/ 제작비 상승
  • 적응형웹 : 레이아웃 크기를 %로 설계
  • 반응형웹 : 사용자의 디바이스 폭크기에 따라 유연하게 표현하는 방식

CSS 단위

  • px : 기본값
  • pt : 출판디자인(호환가능)
  • % : 부모의 크기에 비례한 크기
  • 숫자 : 배수
  • rem : body 태그에 있는 대문자M의 크기를 기준으로 함
  • em : 부모 글자크기에 대한 배율단위
  • vw / wh : 100vh
