레이아웃 : grid

Container 속성

  • display : grid
  • gap : 안쪽 여백
  • grid-template-columns / grid-template-rows : 크기 조정(폭/높이)
  • grid-auto-rows / grid-auto-columns : 지정되지 않은 item의 기본크기
  • justify-items / align-items : cell의 정렬
  • justify-content / align-content : 모든 cell의 정렬

item 속성

  • grid-column / grid-row : 셀의 크기
  • align-self / justify-self : 단독셀의 세로/가로 정렬
  • order : 콘텐츠의 순서를 지정
