가상선택자(가상클래스) / 가상요소

  • :nth-child(n) : 형제번호로 선택
  • :first-child : 첫번째 형제번호 태그를 선택
  • :nth-of-type(n) : 해당태그의 순번으로 선택
  • :link/:hover/:visited/:active : 링크속성/ 마우스 올렸을 때 속성/ 방문했던 주소링크가 걸린경우 / 마우스를 누르고 있을 때
    네이버 다음
  • :focus : 포커스
  • :checked : 체크가 되었을 때 선택되었습니다.
  • :empty : 콘텐츠가 비어있는 태그
  • :not() : 선택자를 제외한 요소
  • 선택자[type=text] : 속성선택자

가상요소

::before / ::after : 가상의 태그를 삽입한 효과
